# RoleLens

`wanted.co.kr`의 채용 카드에서 회사명을 기준으로 공고를 분류하거나 숨길 수 있는 크롬 확장 프로그램입니다.

## 주요 기능

- 회사명을 정확히 일치시키는 방식으로 상태를 저장합니다.
- 공고 카드 위 오버레이에서 빠르게 상태를 바꿀 수 있습니다.
- 관심 없는 공고는 흐리게 표시할 수 있습니다.
- 보고 싶지 않은 회사는 목록에서 숨길 수 있습니다.
- 아직 분류하지 않은 회사의 기본 상태를 설정할 수 있습니다.

## 상태 의미

- `+` : 관심 있음
- `-` : 관심 낮음
- `x` : 숨김

기본 상태는 `+` 또는 `-`만 설정할 수 있습니다.

## 사용 방법

1. 원티드(`wanted.co.kr`)에서 채용 공고 목록을 엽니다.
2. 각 카드에 표시되는 오버레이에서 `+`, `-`, `x` 중 하나를 선택합니다.
3. 설정 페이지에서 회사명을 직접 추가하거나, 저장된 회사를 수정/삭제할 수 있습니다.
4. 분류하지 않은 회사에 대해 기본 상태를 지정할 수 있습니다.

## 설치 방법

```bash
pnpm install
pnpm build
```

빌드 후 생성되는 `dist` 폴더를 크롬의 `확장 프로그램 > 압축해제된 확장 프로그램을 로드`에서 선택하면 됩니다.

## 개발

```bash
pnpm test
```
